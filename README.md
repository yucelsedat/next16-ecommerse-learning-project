# &#x1f6d2; E-Commerce Learning Project

Bu proje, **öğrenme ve pratik amacıyla** geliştirilen bir e-ticaret uygulamasıdır. Proje aşama aşama ilerleyecek ve her aşama tamamlandıkça README güncellenecektir.

> &#x1f3af; Amaç: Modern web teknolojileri ile gerçekçi bir e-ticaret sistemi geliştirerek güçlü bir portföy projesi oluşturmak.

---

## &#x1f9f1; AŞAMA 1 — Core & Foundation

**Bu aşamanın amacı:** Sistemi ayağa kaldırmak

---

### &#x1f510; Admin & Backend

* Admin paneli ve dashboard
* Protected admin routes
* Ürün (Product) CRUD işlemleri
* Kategori yönetimi (CRUD)
* Görsel yükleme sistemi (image upload)

---

### &#x1f310; Frontend (Public)

* Ana sayfa ürün sectionları
* Kategoriye göre ürün listeleme
* Ürün detay sayfası

---

### &#x1f9e0; State Management

* Redux Toolkit kurulumu
* RTK Query ile API yönetimi
* Normalized state (entityAdapter)
